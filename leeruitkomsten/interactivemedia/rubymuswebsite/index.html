UX Website Ruby Mus

Voor de website van Ruby Mus heb ik me bezig gehouden met het maken van de header van de website. Ik heb deze interactief gemaakt door het gebruiken van animaties. Meer over dit project is hier op de development pagina te vinden.

Versie 1 - zonder animaties

Voor de eerste versie van de header heb ik geen gebruik gemaakt van animaties, dit heb ik gedaan om eerst een simpele demo te kunnen laten zien. Na het krijgen van feedback op het einde van sprint 2 ben ik in sprint 3 verder gegaan met het toevoegen van animaties aan de header.

Versie 2 - met animaties
