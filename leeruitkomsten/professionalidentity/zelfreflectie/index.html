Leeruitkomsten Proffesional Identity Zelfreflectie

Zelfreflectie

In deze zelfreflectie leg ik uit wat ik het afelopen semester heb gedaan, wat ik nog wil leren en wat ik tot nu toe heb geleerd. Ook vertel ik hier waarom ik voor de specialisaties Branding en Advanced Development heb gekozen.

Specialisaties

Tijdens het tweede semester van ICT & Media Design had je de keuze uit 5 specialisaties, ik heb voor de specialisaties Branding en Advanced Development gekozen. Ik heb deze specialisaties gekozen om mezelf uit te dagen in de richtingen die mij interesseren.


Branding


Ik heb voor de specialisatie branding gekozen omdat het mij interessant leek om onderzoek te doen naar een bedrijf en erachter te komen hoe een branding in elkaar steekt bij een professioneel bedrijf. Ook vond ik dit een handige keuze omdat ik voor onze groepsopdracht voor Ruby Mus ook bezig was met een brandguide, doordat ik deze specialisatie heb gekozen was het voor mij duidelijker wat er allemaal verwacht wordt in een brandguide. Ook kon ik met de specialisatie twee leeruitkomsten aantonen, namelijk voor mijn research en design leeruitkomst.


Ik heb voor deze opdracht voor het bedrijf Goose Island co. gekozen, dit heb ik expres gedaan omdat dit bedrijf mij aansprak en voor mij nog volkomem onbekend was. Ik ben ontzettend blij dat ik voor deze specialisatie en dit bedrijf heb gekozen, ik heb met veel plezier deze specialisatie uitgevoerd.


Voor de research leeruitkomst van deze specialisatie kun je hier klikken en voor de design leeruitkomst van deze specialisatie kun je hier klikken.


Advanced Development


Deze specialisatie heb ik gekozen voor een flinke uitdaging, ik had zelf nog weinig ervaring in developen. Ik heb development wel altijd heel interessant gevonden en wilde er daarom wel meer mee doen, daarom heb ik voor deze specialisatie gekozen. Ik vond het leuk om met libraries te werken, ik had hier zelf nog nooit mee gewerkt maar er wel al vaker over gehoord.


Ik heb voor deze specialisatie drie libraries uitgewerkt, namelijk Anime.js, AOS (Animate on Scrolling) en Slick. Ik heb voor deze libraries gekozen omdat deze ook nog nuttig konden zijn om in mijn portfolio website te gebruiken. De uitwerking en demo's van deze libraries zijn hier te vinden.


Ik heb na het maken van deze specialisatie meer kennis opgedaan in het developen en dan vooral van JavaScript en de libraries hiervoor, ik ben tevreden met dit resultaat.
