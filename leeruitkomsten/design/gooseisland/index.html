Leeruitkomsten Design Goose Island

Goose Island

Voor de specialisatie Branding heb ik een merkonderzoek gedaan naar brouwerij Goose Island met gebruik van de brand canvas. Ik heb dit gevisualiseerd in Adobe InDesign zodat ik het voor de leeruitkomsten research en design kon gebruiken. Op deze pagina zal ik vooral vertellen over het design van het document, over het research gedeelte van het project kun je hier meer lezen.

Opdracht

De opdracht voor deze specialisatie was het doen van een brand research door middel van een brand canvas of brand identity prism. Ik heb gekozen om voor deze opdracht met de brand canvas methode te werken omdat deze mij het meest geschikt leek. Ook was het de bedoeling dat alle inhoud van het document volgens de APA-richtlijnen verwerkt zou worden. Omdat ik voor dit project een mooi visueel plaatje wilde maken heb ik dit project uitgewerkt in Adobe InDesign, op deze manier kon ik ook mijn design leeruitkomst aantonen. Ik heb er voor gekozen om een Brandguide te maken, dit heb ik gedaan omdat ik deze ook nog een wilde maken voor het project van Ruby Mus. Op deze manier kon ik hier alvast gebruik van maken.

Stijl

Omdat Goose Island natuurlijk al een bestaand bedrijf is met een eigen huisstijl heb ik in mijn document geprobeerd zoveel mogelijk hun huisstijl aan teh houden. Zo heb ik gebruik gemaakt van hun kleuren, fonts en andere elementen als logo's. Deze stijl is door het volledige document terug te vinden waardoor het overkomt alsof het ook echt bij hun brand hoort.

Feedback

De feedback die ik voor het design van mijn document heb gekregen was positief, de docent en mijn medegroepsleden vonden het een overzichtelijk en mooi vormgegeven document.


Van de eindbeoordelaar kreeg ik de opmerking: "Netjes gedaan Sam, mooi vormgegeven ook."


Kortom heb ik dus positieve feedback gekregen op mijn geleverde eindproduct, ik ben zelf tevreden met het resultaat en vond het een leuke opdracht om te maken.

Wat heb ik geleerd?

Ik heb meer geleerd over het werken met Adobe InDesign, ook heb ik meer geleerd over wat er inhoudelijk wordt verwacht in een brandguide. Ook vond ik het leuk om dit te maken over een bedrijf wat voor mij nog totaal onbekend was, ik heb hierdoor dus zeker ook meer praktijk ervaring gekregen met onderzoek doen naar een bedrijf.

Wat kon beter?

Ik had eerder feedback kunnen vragen, ik heb dit niet gedaan omdat ik er zo lekker mee bezig was waardoor ik het eigenlijk eerst wilde afmaken om zo meteen een mooi eindresultaat te kunnen laten zien.
