Mediaverkenning Ruby Mus

In sprint 0 zijn wij begonnen met het maken van een mediaverkenning voor Ruby Mus. Hierin zijn dingen als meer informatie over Ruby Mus, haar stijl, inspiratie en haar doelgroep te vinden. Voor dit project ben ik verantwoordelijk geweest voor het maken van de covers en doelgroep pagina's.

Brandguide Ruby Mus

In sprint 1 heb ik voor Ruby Mus een brandguide gemaakt in Adobe InDesign, hierin is alles over haar merk stijl terug te vinden.

Website Ruby Mus

In sprint 1 heb ik een website design gemaakt voor Ruby Mus haar website, dit heb ik gedaan in Adobe XD. Hierna heb ik ook nog een gezamelijk design gemaakt met mijn andere groepsleden.

Goose Island

Voor mijn specialisatie Branding heb ik met de brand canvas methode een brand research gedaan naar bierbrouwerij Goose Island, dit heb ik gevisualiseerd in Adobe InDesign.

Fotoshoot Ruby Mus

Tijdens sprint 2 hebben ik en mijn groepsleden een fotoshoot gehouden met Ruby Mus, het doel hiervan was om proffesionele beelden te creëren die we konden gebruiken voor haar website en sociale media. Na de shoot heb ik deze foto's bewerkt in Adobe Photoshop.
