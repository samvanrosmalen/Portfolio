Leeruitkomsten Development Image Slider

Image Slider

Voor mijn eerste POC ben ik een Image Slider gaan maken, hiervoor heb ik gebruik gemaakt van de opdracht en uitleg op canvas. Met het maken van deze opdracht heb ik wat basis kennis opgedaan van JavaScript.

Structuur

Ik heb in HTML 2 foto’s gezet deze heb ik vervolgens met CSS-code over elkaar heen gezet. Door alles duidelijke class names te geven was het makkelijk om alles weer terug te halen in JavaScript.

Opmaak

Ik heb voor deze POC vrijwel niks aan de opmaak gedaan omdat de POC meer om de functie gaat en niet om de opmaak. Het enige wat ik heb gedaan is een titel boven de foto gezet.

Functies

In JavaScript heb ik de foto’s die ik wilde gebruiken voor mijn slider in een array gezet, vervolgens heb ik een functie gemaakt waardoor de front image en back image over elkaar heen faden. Als bijvoorbeeld de front image in beeld is veranderd de back image op de achtergrond naar een andere foto uit de array.

Wat heb ik geleerd?

Ik heb geleerd om met JavaScript op een best makkelijke manier een image slider te maken, hierdoor is mijn kennis van JavaScript weer iets omhooggegaan.

Wat kon beter?

Ik had meer tijd aan de opmaak kunnen besteden waardoor de image slider er beter uitzag.
