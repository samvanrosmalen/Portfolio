Leeruitkomsten Development Portfolio

Portfolio (Dev)

Ik heb dit semester hard gewerkt aan het maken en bijhouden van dit portfolio. Ik ben begonnen met het maken van een design in Adobe XD, om ook voor development op schema te blijven heb ik sommige pagina's ook meteen uitgewerkt na het maken van het design. Nadat ik mijn design heb laten zien aan verschillende leraren ben ik me gaan focussen op het programmeren van de website. Hiervoor heb ik gebruik gemaakt van HTML, CSS, Sass, JavaScript en JavaScript libraries.

Structuur

Omdat ik een goede structuur wilde voor mijn portfolio heb ik alles ingedeeld in mappen, zo heb ik bijvoorbeeld de map leeruitkomsten met daarin de map development met daarin de map portfolio. Op deze manier heb ik een goed overzicht wat waar bij hoort. Om mijn HTML code netjes te houden heb ik geprobeerd alles zo makkelijk mogelijk te programmeren en logische class en id names te geven.

Opmaak

Voor de opmaak heb ik gebruik gemaakt van CSS. Door logische namen te geven heb ik ervoor gezorgd dat alles overzichtelijk bleef voor mij. Voor het positioneren heb ik op aanraden van de docenten vooral Flexbox gebruikt, de volgende keer wil ik ook leren werken met Bootstrap.

Functies

Ook heb ik gebruik gemaakt van JavaScript, dit heb ik gebruikt voor mijn geannimeerde logo, de back to top button en custom cursor. Ook heb ik in mijn portfolio gebruik gemaakt van mijn POC Advanced Development door het gebruik van de AOS, Slick en Animate.css libraries

Interactie

Door het gebruiken van animaties en geannimeerde logo's heb ik mijn portfolio website een betere UX gegeven. Hiervoor heb ik de animate on scroll (AOS) en animate.css library gebruikt.

Wat heb ik geleerd?

Tijdens het maken van mijn portfolio heb ik meer geleerd van het werken met JavaScript, ook heb ik flink wat stappen gemaakt met het programmeren in CSS en voelt het voor mijn nu allemaal wat logischer.
