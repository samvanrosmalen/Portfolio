Advanced Development

Ik heb gekozen om de keuzemodule advanced development te kiezen, dit heb ik gedaan omdat het me leuk en nuttig leek om meer te leren over JavaScript en libraries. Tijdens de eerste week ben ik op internet gaan zoeken naar verschillende libraries, hieruit heb ik er 10 gekozen die mij leuk en nuttig leken. Voor deze 10 libraries heb ik een stukje getypt over wat ze doen en hoe ik ze kan gebruiken (zie pdf document). Vervolgens heb ik hieruit 3 libraries gekozen die ik uit ben gaan werken.

Anime.js

De eerst library die ik uit ben gaan werken is Anime.js, hiermee kun je op een makkelijke manier animaties toevoegen aan je website. Voor het maken van een demo voor deze library ben ik op de website van Anime.js gaan kijken wat er allemaal mogelijk is met deze library, hierna ben ik wat dingetjes gaan testen. Ook zag ik een tutorial hoe je een effect kon maken met een tekst of naam, ik heb deze tutorial gevolgd en hier is een mooi resultaat uitgekomen.

AOS

De tweede library die ik heb uitgewerkt is AOS (Animate on scroll), met deze library kun je animaties aan je website toevoegen die gekoppeld zijn aan je scroll gedrag. Hiermee kun je dus zorgen dat iets pas infade als je op een bepaald stuk van de website zit. Toen ik deze library tegenkwam wist ik al meteen dat ik deze ook wil toepassen op mijn portfolio website.

Slick

Als derde library heb ik gekozen om Slick uit te werken, met deze library kun je makkelijk een slider maken voor bijvoorbeeld foto's. Ook bij deze library dacht ik al dat ik deze nog ergens kon gaan toepassen op mijn portfolio website. Ik heb geleerd dat slick echt een hele fijne en makkelijke manier is om een slider te maken, je hebt veel opties voor het aanpassen van snelheden, autoplay, dots etc.

Toepassing Portfolio

In mijn portfolio heb ik 2 libraries toegepast: AOS en Slick. Ik heb AOS toegepast op alle verschillende leeruitkomsten pagina's, hierdoor is de website wat meer interactief en het geeft een mooi effect. Slick heb ik toegepast op de fotoshoot pagina van Ruby Mus, hier laat ik wat resultaten zien in een foto slider.

Wat heb ik geleerd?

Ik heb tijdens het maken van deze keuzemodule geleerd wat libraries zijn en hoe ik deze kan inladen en toepassen. Zo heb ik de basis geleerd van werken met de libraries Anime.js, AOS en Slick.

Wat kon beter?

Ik had eerder aan deze keuzemodule kunnen werken, doordat ik het druk had met andere projecten heb ik dit project naar achteren geschoven. Ik had dan misschien wat mooiere demo's kunnen maken.
